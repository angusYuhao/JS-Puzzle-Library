API DESCRIPTION

Library Functions:

constructor

ImagePuzzle(puzzleType)

Description:

Create a new puzzle object, puzzle type is given in 
puzzleType

Arguments:

puzzleType
 : can be the following: 
"grid"
 
"scatter"
 
"scramble"
 
"cutout"


User Functions

bindImage(imgURL)

Description:

Bind an image to the puzzle

Arguments:

imgURL
 : a string such as 
"./image.jpg"



bindBackgroundImage(imgURL)

Description:

Bind a background image to the puzzle canvas

Arguments:

imgURL
 : a string such as 
"./image.jpg"



createGridCanvas(parentElement, numRows, numCols, gapSize, titleBarColor="cornflowerblue", titleTextColor="white", titleText="", slotColor="grey", slotBorderRadius="0px", backgroundColor="white", canvasBorderRadius="0px" )

Description:

Create the main grid canvas, can only be used if puzzle type is "grid"

Arguments:

parentElement
 : the parent element that wraps the canvas (DOM element), width must be set explicitly: 100%, 50px, etc. 

numRows
 : number of rows in the puzzle (integer) 

numCols
 : number of columns in the puzzle (integer) 

gapSize
 : gap size between puzzle slots on the canvas (integer) 

titleBarColor
 : color of the title bar (string) 

titleTextColor
 : color of the title text (string) 

titleText
 : text to display on title bar (string) 

slotColor
 : color of each puzzle slot on the canvas (string) 

slotBorderRadius
 : border radius of each puzzle slot on the canvas (string) 

backgroundColor
 : background color of the canvas (string) 

canvasBorderRadius
 : border radius of the canvas (string)



createScatterCanvas(parentElement, slotsArray, titleBarColor="cornflowerblue", titleTextColor="white", titleText="", slotColor="grey", slotBorderRadius="0px", backgroundColor="white", canvasBorderRadius="0px")

Description:

Create the main scatter canvas, can only be used if puzzle type is "scatter"

Arguments:

parentElement

const slots = [

                { top: "40%", left: "74.9%", height: "9%", width: "20.4%"},
                { top: "74.8%", left: "59.9%", height: "9.1%", width: "16.6%"}
            ]
 

gapSize
 : gap size between puzzle slots on the canvas (integer) 

titleBarColor
 : color of the title bar (string) 

titleTextColor
 : color of the title text (string) 

titleText
 : text to display on title bar (string) 

slotColor
 : color of each puzzle slot on the canvas (string) 

slotBorderRadius
 : border radius of each puzzle slot on the canvas (string) 

backgroundColor
 : background color of the canvas (string) 

canvasBorderRadius
 : border radius of the canvas (string)

createCutoutCanvas(canvasParentElement, cutoutsArray, trayParentElement, numRows, numCols, gapSize, orderMap, cTitleBarColor="cornflowerblue", cTitleTextColor="white", cTitleText="", cSlotColor="grey", cBackgroundColor="white", canvasBorderRadius="0px", slotColor="grey", backgroundColor="white", trayBorderRadius="0px")

Description:

Create the main grid canvas and the tray, can only be used if puzzle type is "cutout"

Arguments:

canvasParentElement
 : the parent element that wraps the canvas (DOM element), width must be set explicitly: 100%, 50px, etc. 

cutoutsArray
 : an array containing the shapes' vertices, for 3 irregular shapes, this might look like: 

const cutoutsArray = [
                    [
                        { x: 1800, y: 700 },
                        { x: 2160, y: 260 },
                        { x: 2520, y: 700 }
                    ],
                    [
                        { x: 1600, y: 2000 },
                        { x: 1600, y: 2480 },
                        { x: 2720, y: 2480 },
                        { x: 2720, y: 2000 }
                    ],
                    [
                        { x: 1600, y: 1000 },
                        { x: 1800, y: 1380 },
                        { x: 2220, y: 1780 },
                        { x: 2720, y: 1400 }
                    ]
                ]
 

trayParentElement
 : the parent element that wraps the tray (DOM element), width must be set explicitly: 100%, 50px, etc. 

numRows
 : number of rows in the tray (integer) 

numCols
 : number of columns in the tray (integer) 

gapSize
 : gap size between puzzle pieces in the tray (integer) 

orderMap
 : a 2D array containing the order the pieces are displayed in the tray, for a 3 x 3 tray, this might look like: 

const orderMap = [
                    [8, 1, 3],
                    [2, 5, 4],
                    [0, 7, 6]
                ]
 

cTitleBarColor
 : color of the title bar (string) 

cTitleTextColor
 : color of the title text (string) 

cTitleText
 : text to display on title bar (string) 

cSlotColor
 : color of each puzzle slot on the canvas (string) 

cBackgroundColor
 : background color of the canvas (string) 

canvasBorderRadius
 : border radius of the canvas (string) 

slotColor
 : color of each puzzle slot in the tray (string) 

backgroundColor
 : background color of the tray (string) 

canvasBorderRadius
 : border radius of the tray (string)



createScatterMatchings(slotArray, imgURLArray)

Description:

Match each slot on the canvas to an image URL. Below, first slot is matched with "./image3_engine.jpg" and etc.

Arguments:

slotArray
 : an array containing the IDs of the slot (integers from 0), this might look like: 

let slotArray = [0, 1, 2]
 

imgURLArray
 : an array containing the URLs of the images (string), this might look like: 

let imgURLArray = ["./image3_engine.jpg", "./image3_command.jpg", "image3_lunar.jpg"]

displayTitle

titleBarColor

slotColor

slotBorderRadius

backgroundColor
 : background color of the tray (string) 

trayBorderRadius
 : border radius of the tray (string)


Event Handlers

"puzzleUpdated"

Description:

fires every time puzzle is updated, contains information about the puzzle

Details:

detail.pieceStatus
 : Is the puzzle piece just dropped in the correct slot? (boolean) 

detail.puzzleStatus
 : Is the puzzle complete? (boolean) 

detail.targetSlotID
 : The ID of the slot the puzzle piece was dropped into (integer) 

detail.action
 : can be "drop", "switch", "initial" based on user's actions

Use Example:

canvasContainer.addEventListener('puzzleUpdated', function (e) {
                    if (e.detail.puzzleStatus) {
                        console.log("puzzle completed!")
                    }
                })
